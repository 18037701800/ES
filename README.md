# ES

## 框架文件

整个框架核心四个文件，所有文件加起来放在一起总行数不超过400 行 

```
protected
|--lib
  |--es.php //启动文件
  |--controller.php //控制器文件
  |--model.php //模型文件
  |--view.php //视图引擎
```
